# Backend Auth CRUD Project

## Description
This project is a scalable REST API built using Node.js, Express, and MongoDB.
It implements JWT-based authentication and protected CRUD operations, along with a simple frontend to demonstrate API usage.

---

## Features
- User Registration
- User Login with JWT Authentication
- Password Hashing using bcrypt
- Protected APIs using middleware
- CRUD operations (Tasks)
- MongoDB database integration
- Simple frontend UI for testing APIs

---

## Tech Stack
- Backend: Node.js, Express.js
- Database: MongoDB
- Authentication: JWT
- Frontend: HTML, JavaScript
- Tools: Git, GitHub, VS Code
